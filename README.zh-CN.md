# Wow-Lyric

## 简介
![输入图片说明](.gitee/%E5%9B%BE%E7%89%87.png)
华丽的歌词动画

如果用过苹果的音乐app，就会觉得很熟悉。

## 使用

### 快速尝试:

[github.io](https://superkenvery.github.io/wow-lyric/)

[腾讯云](http://my-bucket-1252360629.cos-website.ap-guangzhou.myqcloud.com/)

### 自行搭建此网页:

1. 克隆或者下载此仓库
2. 用浏览器打开 `index.html`
3. 上传歌曲文件、歌词文件和专辑图文件
4. 点击 `开始 Start`按钮
